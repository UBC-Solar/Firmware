To run the serial GUI (Windows only):
1) Install the lastest version of Python 3 from https://www.python.org/downloads/. Before clicking install now, check the box "add Python 3.6 to PATH"
2) Go to C:\Program Files (x86)\Python35-32\
3) Hold down shift and right click the directory "Scripts." click on "run window on command prompt"
4) Type "python -m pip install pyserial"
5) Go to the start menu (Cortona) and search for "device manager"
6) Plug in wireless module. Click "Ports" and find the wireless module. Remember the COM number
7) Double click serial_mod.py in the telemetry node directory. Type in the COM number
